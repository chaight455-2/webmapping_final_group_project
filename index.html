About this app

UW-Madison Amenities helps students find everyday essentials — microwaves, refrigerators, vending machines, public computer labs, and printers/copy machines — across academic buildings on campus. Use it to plan a quick lunch reheat between classes or find a printer when you need one.

How to use

  • Search for a building by name using the bar at the top.
  • Filter by amenity type using the Filters menu in the top-left. Only buildings with the selected amenities will show colored dots.
  • Tap a building on the map to see its open/closed status, hours, and a list of available amenities.
  • Tap an amenity row in the building popup to see the floor and location of each one inside the building.
  • Find yourself by tapping the location button in the bottom-right corner to show your current position on the map.

Open/closed status

Each building shows a green Open or red Closed badge based on the building's hours. Tap Hours in the popup to see the full weekly schedule.

Want to plan ahead? Use the Check open at day and time picker inside the Filters menu to see which buildings will be open at a future day or time. Choose Current time in the time dropdown to reset to the current time.
